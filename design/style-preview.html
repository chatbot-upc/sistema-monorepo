Fichas técnicas

tipografías · paletas · cuándo funciona cada uno

A · Editorial Académico
Fraunces (display serif variable, italic para énfasis) + Inter (body) + JetBrains Mono (datos).
Paleta: crema #FAF7F2 · tinta #1A1A1A · terracota #B8553A · ochre para escalado.

Funciona si: quieres que tu tesis destaque por cuidado editorial, peso tipográfico y seriedad institucional no corporativa. El hilo de conversación se lee como una entrevista publicada.
No funciona si: necesitas sensación de tiempo real o alta densidad.

B · Data-Dense Técnico
Inter (display/UI, negative tracking) + JetBrains Mono (metadata protagonista).
Paleta: negro casi puro #0A0A0B · lima eléctrico #C4F547 · ámbar #F5A623 para alertas.

Funciona si: quieres que el CRM se sienta como herramienta de operador/observabilidad. Mucha metadata visible (confianza, p95, timestamps precisos). Dark-first.
No funciona si: el admin no es técnico — puede sentirse frío o abrumador.

C · Warm Peruano Moderno
Instrument Serif (display) + Inter (body) + IBM Plex Mono (datos).
Paleta: crema #F5EFE6 · obsidiana #1F1B17 · rojo UPC #A01E2D · detalles de doble línea.

Funciona si: quieres identidad peruana/institucional clara pero moderna. El rojo UPC comunica pertenencia sin caer en folclor.
No funciona si: el rojo compite demasiado con los pills de estado y puede saturar en sesiones largas.
